JavaScript 计时器

  • 计时事件

    JavaScript一个设定的时间间隔之后来执行代码，而不是在函数被调用后立即执行，我们称之为计时事件

  • 关键字

    关键字 | 简介 ---|--- setInterval | 间隔指定的毫秒数不停的执行指定的代码（不停地重复执行） clearInterval | 停止setInterval()方法 setTimeout | 暂停指定的毫秒数后执行指定的代码（只执行一次） clearTimeout | 停止执行setTimeout()方法

  • setInterval(重复执行的代码，毫秒数)

    重复执行同一个函数，重复时间间隔为第二个参数的毫秒数（1秒=1000毫秒）

    示例（每隔一秒打印当前时间）：

<p>页面上显示时钟：</p>
<p id="time"></p>
<script>
    function printTime(){
      var d = new Date();
      var h= d.getHours();//时
      var m= d.getMinutes();//分
      var s= d.getSeconds();//秒
      document.getElementById("time").innerHTML= h+":"+m+":"+s;
    }
    var t = setInterval(printTime,1000);
</script>

  • clearInterval(setInterval计时器)

    停止setInterval()计时器

    实例（点击按钮停止时钟）：

<p>页面上显示时钟：</p>
<p id="time"></p>
<button onclick="myStopFunction()">点击停止时钟</button>
<script>
    function printTime(){
      var d = new Date();
      var h= d.getHours();//时
      var m= d.getMinutes();//分
      var s= d.getSeconds();//秒
      document.getElementById("time").innerHTML= h+":"+m+":"+s;
    }
    var t = setInterval(printTime,1000);
    function myStopFunction(){
        clearInterval(t);
    }
</script>

  • setTimeout(执行的代码，毫秒数)

    一定毫秒数后执行函数

    实例（点击按钮后3秒弹出对话框）：

<p>点击按钮，在等待 3 秒后弹出 "Hello"。</p>
<button onclick="myFunction()">点我</button>
<script>
    function myFunction(){
        setTimeout(function(){alert("Hello")},3000);
    }
</script>

  • clearTimeout(setTimeout计时器)

    停止setTimeout计时器

    当setTimeout计时器放在要执行的函数里面时，就会产生与setInterval计时器一样的效果，会不停的重复执行这个函数，这时就可以用clearTimeout()来停止计时器

    示例（还是上面的时钟示例）：

<p>页面上显示时钟：</p>
<p id="time"></p>
<button onclick="myStopFunction()">点击停止时钟</button>
<script>
    var t;
    function printTime(){
      var d = new Date();
      var h= d.getHours();//时
      var m= d.getMinutes();//分
      var s= d.getSeconds();//秒
      document.getElementById("time").innerHTML= h+":"+m+":"+s;
      t = setTimeout(printTime(),1000);
    }
    function myStopFunction(){
        clearTimeout(t);
    }
</script>
