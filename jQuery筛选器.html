jQuery筛选器

  • 筛选器指的是在已经通过选择器选中了元素后，在此基础上进一步选择。

关键字|简介 ---|--- first()|第1个元素 last()|最后1个元素 eq(num)|第num个元素 parent()|选取最近的一个父元素 parents()|选取所有的祖先元素 children()|子元素 find()|后代元素 siblings()|同级(同胞)元素

  • 示例（first()，last()，eq(num)）：

    first()--------第1个元素

    last()--------最后1个元素

    eq(num)----第num个元素

<style>
    .pink{background-color:pink;}
</style>
<body>       
    <script>
        $(function() {
            $("#b1").click(function() {
                ·$("div").first().toggleClass("pink");
            });
            $("#b2").click(function() {
                $("div").last().toggleClass("pink");
            });
            $("#b3").click(function() {
                $("div").eq(4).toggleClass("pink");
            });
        });
    </script>
    <button id="b1">切换第1个div背景色</button>
    <button id="b2">切换最后1个div背景色</button>
    <button id="b3">切换第5个div背景色</button>
    <br>
    <br>
    <div>DIV 1</div>
    <div>DIV 2</div>
    <div>DIV 3</div>
    <div>DIV 4</div>
    <div>DIV 5</div>
    <div>DIV 6</div>
</body>

  • 示例（parent()，parents()）

    parent()-----选取最近的一个父元素

    parents()----选取所有的祖先元素

<style>
    div{padding:20px;}
    #grandParentDiv{background-color:pink;}
    #parentDiv{background-color:green;}
    #currentDiv{background-color:red;}
    .b{border: 2px solid black;}
</style>
<body>
    <script>
        $(document).ready(function(){
            $("#b1").click(function(){
                $("#currentDiv").parent().toggleClass("b");
            });
            $("#b2").click(function(){
                $("#currentDiv").parents().toggleClass("b");
            });
        });
    </script>
    <button id="b1">改变parent()的边框</button>
    <button id="b2">改变parents()的边框</button>
    <div id="grandParentDiv" >
        祖先元素
        <div id="parentDiv">
            父元素
            <div id="currentDiv">当前元素</div>  
        </div>
    </div>
</body>

  • 示例（children()，find()）：

    children()-----子元素

    find()-----------后代元素 

<style>
    div{padding:20px;}
    .grandChildrenDiv{background-color:pink;}
    .childrenDiv{background-color:green;margin:10px;}
    #currentDiv{background-color:red;}
    .b{border: 2px solid black;}
</style>
<body>
    <script>
        $(document).ready(function(){
            $("#b1").click(function(){
                $("#currentDiv").children().toggleClass("b");
            });
            $("#b2").click(function(){
                $("#currentDiv").find("div").toggleClass("b");
            });
        });
    </script>
    <button id="b1">改变children()的边框</button>
    <button id="b2">改变find()的边框</button>
    <div id="currentDiv" >
        当前元素
        <div class="childrenDiv">
            子元素1
            <div class="grandChildrenDiv">后代元素n</div>  
        </div>
        <div class="childrenDiv">
            子元素2
            <div class="grandChildrenDiv">后代元素n</div> 
        </div>
    </div>
</body>

  • 示例（siblings()）

    siblings()------同级(同胞)元素

<style>
    div{padding:20px; background-color:pink; margin:10px;}
    div#parentDiv{background-color:green;}
    div#currentDiv{background-color:red;}
    .b{border: 2px solid black;}
</style>
<body>
    <script>
        $(document).ready(function(){
            $("#b1").click(function(){
                $("#currentDiv").siblings().toggleClass("b");
            });
        });
    </script>
    <button id="b1">给同级加上边框</button>
    <div id="parentDiv" >
        父元素
        <div id="currentDiv">当前元素</div>
        <div>同级元素</div>
        <div>同级元素</div>
    </div>
</body>
