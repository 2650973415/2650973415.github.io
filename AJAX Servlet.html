AJAX + Servlet与数据库交换数据

AJAX = Asynchronous JavaScript and XML (异步的JavaScript 和 XML)。

AJAX 不是一种新的编程语言，而是一种使用现有标准的新方法。

AJAX 是与服务器交换数据兵更新部分网页的艺术，再不重新加载整个网页的情况下。

  • 创建 XMLHttpRequest 对象 所有现代浏览器均支持 XMLHttpRequest 对象（IE5 和 IE6 使用 ActiveXObject）。 XMLHttpRequest 用于在后台与服务器交换数据。这意味着可以在不重新加载整个网页的情况下，对网页的某部分进行更新。
var xmlhttp;
if (window.XMLHttpRequest)
{
    //  IE7+, Firefox, Chrome, Opera, Safari 浏览器执行代码
    xmlhttp=new XMLHttpRequest();
}
else
{
    // IE6, IE5 浏览器执行代码
    xmlhttp=new ActiveXObject("Microsoft.XMLHTTP");
}

  • 向服务器发送请求请求

    open() 方法的第一个值是请求的类型：GET 或 POST。第二个值是url：文件在服务器上的位置。第三个值是是否是异步：true（异步）或 false（同步）

xmlhttp.open("GET","/users/getShi?pid=" + Math.random(),true);
xmlhttp.send();

  • 服务器响应

    获取来自服务器的响应，使用 XMLHttpRequest 对象的 responseText 或 responseXML 属性。

    属性 | 描述 ---|--- responseText | 获得字符串形式的响应数据。 responseXML | 获得 XML 形式的响应数据。

    只介绍responseText属性使用：

    document.getElementById("myDiv").innerHTML=xmlhttp.responseText;

  • onreadystatechange 事件

    在 onreadystatechange 事件中，我们规定当服务器响应已做好被处理的准备时所执行的任务。 当 readyState 等于 4 且状态为 200 时，表示响应已就绪：

xmlhttp.onreadystatechange=function(){
    if (xmlhttp.readyState == 4 && xmlhttp.status == 200){
        document.getElementById("myDiv").innerHTML=xmlhttp.responseText;
    }
}

jsp与ajax部分：

<body>
    <select id="sheng">
        <c:forEach var="entry" items="${requestScope.diQu}">
            <option value="${entry.key}">${entry.value}</option>
        </c:forEach>

    <select id="shi">

    </select>
    <script src="../js/jquery-3.2.1.min.js">adsa</script>
    <script>
        $(document).ready(function () {
            $("#sheng").change(function () {
                var xmlhttp;
                if (window.XMLHttpRequest){
                    // IE7+, Firefox, Chrome, Opera, Safari 浏览器执行代码
                    xmlhttp=new XMLHttpRequest();
                }
                else{
                    // IE6, IE5 浏览器执行代码
                    xmlhttp=new ActiveXObject("Microsoft.XMLHTTP");
                }
                xmlhttp.onreadystatechange=function(){
                    if (xmlhttp.readyState==4 && xmlhttp.status==200){
                        document.getElementById("shi").innerHTML=xmlhttp.responseText;
                    }
                }
                xmlhttp.open("GET","/users/getShi?pid="+this.value,true);
                xmlhttp.send();
            })
        });
    </script>
</body>

servlet部分：

public class UserIndexServlet extends HttpServlet {
    IUserService iUserService = new UserServiceImpl();
    protected void doGet(HttpServletRequest req, HttpServletResponse resp) throws ServletException, IOException {String str = "";
        HashMap<String,String> map = null;
        try {
            map = iUserService.getSheng();
        } catch (UserException e) {
            e.printStackTrace();
        }
        req.setAttribute("diQu",map);
        this.getServletConfig().getServletContext().getRequestDispatcher("/users/index.jsp").forward(req,resp);
    }
}

service部分：

public class UserServiceImpl implements IUserService {
    IUserDao iUserDao = new UserDaoImpl();
    @Override
    //getSheng()获取省级地区
    public HashMap<String, String> getSheng() throws UserException {
        return iUserDao.getSheng();
    }
    @Override
    //getShi()获取市级地区
    public HashMap<String, String> getShi(int pid) throws UserException {
        return iUserDao.getShi(pid);
    }
}

dao部分：

public class UserDaoImpl implements IUserDao {
    @Override
    //getSheng()获取省级地区
    public HashMap<String, String> getSheng() throws UserException {
        return selectDB("select * from china where pid=0 and id!=0","id","name");
    }
    @Override
    //getShi()获取市级地区
    public HashMap<String, String> getShi(int pid) throws UserException {
        return selectDB("select * from china where pid="+pid,"id","name");
    }
    //查询数据（指定两个字段）
    public HashMap<String, String> selectDB(String selectSql,String field1,String field2) throws UserException {
        HashMap<String, String> map = new HashMap<>();
        Connection connection = MysqlUtil.getInstace();
        try {
            PreparedStatement preparedStatement = connection.prepareStatement(selectSql);
            ResultSet resultSet = preparedStatement.executeQuery();
            while (resultSet.next()){
                String id = resultSet.getInt(field1)+"";
                String name = resultSet.getString(field2);
                map.put(id,name);
            }
        } catch (Exception e) {
            throw new UserException("查询数据异常："+selectSql);
        }
        return map;
    }
}
  • jQuery AJAX实现：

$.ajax()

$.ajax({
    type:"Get",
    url:"/users/getShi?pid="+this.value,
    success: function (html) {
        document.getElementById("shi").innerHTML=html;
    }
});

$.get("/userInfo","pid="+pid,function (info) {
    document.getElementById("shi").innerHTML=info;

$.post("/userInfo","pid="+pid,function (info) {
    document.getElementById("shi").innerHTML=info;
